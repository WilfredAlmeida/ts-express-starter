# ts-express-starter
This repo is a TypeScript + Express Starter.

This is built using `pnpm`, but you can use any package manager you like.

## Getting Started

1. Clone the repo.
2. Run `pnpm install`
3. Run `pnpm run dev`

That's all you need to get started. Edit the configs as you need in `package.json` and `tsconfig.json`
